# Blinkit-sales-dashboard

**Blinkit Sales Analysis Dashboard**

This project was developed using Power BI to analyze Blinkit sales data and outlet performance.

Tools Used

- Power BI
- Excel

Features

- KPI cards for Total Sales, Avg Sales, Avg Rating
- Outlet performance analysis
- Item category insights
- Interactive filters and charts

Dashboard Insights

- Tier 3 outlets generated highest sales
- Regular fat products contributed more revenue
- Supermarket Type1 had highest sales performance
